How might we solve this using the small changes method.\
First add a variable h which represents a small change to the input x.\ 
Then after expanding out the function you just need the term linear with h.\

Since we only need one h,\
we can choose one (x plus h), pull out the h (discarding the x)\
and for the rest of the product terms we discard the h (keeping the x).\
This gives h times x to the power of n minus one.\

The solution we want is the sum of all the ways we can do this.\
The coefficient in front of this simplified sum is call n choose 1.\
It evaluates all the ways we can choose one item from n items\
and it is equal to n.\

This means the derivative of\ 
x to the power of n \
is n times x tp the power of n minus one.
