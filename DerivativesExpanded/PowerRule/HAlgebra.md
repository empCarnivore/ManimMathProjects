How might we solve this using the small changes method.\
First add a variable h which represents a small change to the input x.\ 
Then after expanding out the function you just need the term linear with h.

Looking at this product of arbitrary size\
you may be wondering how are we supposed to expand this out.\
And the key to this is that you actually don't need to.\
Since we only care about the linear to h terms we can ignore everything else.

These next steps is sort of like the foil rule if you have ever taken US algebra.\
Only we have more terms in the product and we just ignore every way that doesn't have a single h.\
Take the first h.\ 
Then multiply it by all the other x terms\ 
excluding the x it's summed with and the other h's.\
Then take the second h and do the same.\
Add all these together, and you'll get the term linear with h.

For this problem all corresponding terms produce the same result. \
h times x to the power of n minus one\
And since there are n terms in the product,\ 
Then the final result is n times that.

This means the derivative of\ 
x to the power of n \
is n times x to the power of n minus one.
